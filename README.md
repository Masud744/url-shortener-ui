# URL Shortener UI

A minimal glassmorphism-based frontend for a webhook-powered URL shortener.

This UI allows users to submit long URLs and receive shortened URLs with visual feedback.

---

## Live web

https://masud744-url-shortener.netlify.app/

## Demo Video

[![Demo Video](https://img.youtube.com/vi/_utEkyYTlkE/0.jpg)](https://youtu.be/_utEkyYTlkE)


## Features

- Dark glassy UI
- URL input form
- Loading indicator
- Confetti & success sound
- Lightweight and responsive

---

## Tech Stack

- HTML
- CSS
- JavaScript

---

## Backend

This frontend is connected to an n8n-powered backend:

https://github.com/Masud744/n8n-url-shortener

The backend handles:

- Webhook requests
- URL shortening logic
- API integration
